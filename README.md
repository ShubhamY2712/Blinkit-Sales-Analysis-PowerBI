# Blinkit Sales & Operational Analysis Dashboard

## 📊 Project Overview
This Power BI dashboard provides a detailed breakdown of Blinkit's sales performance, focusing on how outlet location (Tier 1, 2, 3), outlet size, and item characteristics affect total revenue.

## 🚀 Key Metrics 
* **Total Sales:** $1.20M (reflecting the overall revenue in the dataset).
* **Average Sales:** $141 per transaction.
* **Item Count:** 8,523 unique items tracked.
* **Customer Satisfaction:** Average Rating of 3.9/10.

## 🛠️ Analysis Dimensions
* **Outlet Performance:** Analysis of sales across **Tier 1, Tier 2, and Tier 3** locations.
* **Outlet Size:** Comparison between **Small, Medium, and High** outlet footprints.
* **Item Details:** Filtering by **Fat Content** (Low Fat vs. Regular) and **Item Type**.
* **Establishment Insights:** Tracking sales growth based on the **Year of Establishment**.
